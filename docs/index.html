    Authentication

    Method Description
    GET Fetch resource
    POST Submit data

    To authenticate with mafia you need to pass the user token with your POST request

    Getting the cookie

    Make a POST request to https://mafia.gg/api/user-session with this payload 

    1
    { "login" : username, "password": password }
